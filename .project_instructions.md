# CashFlowIQ - Full SaaS Platform - Project Instructions\n\n## Project Overview\nCashFlowIQ is an enterprise AI-powered SaaS cash flow forecasting platform for SMBs. The platform has been fully built through 5 phases:\n- **Phase 1-2 (Complete)**: MVP with Stripe billing, Plaid integration, BullMQ background jobs\n- **Phase 3 (Complete)**: Multi-account management, advanced forecasting, PDF/Excel exports, API access\n- **Phase 4 (Complete)**: Role-based access control (RBAC), audit logging, shared dashboards, team billing\n- **Phase 5 (Complete)**: Performance caching, GDPR compliance, integrations (QB/Wave/Xero), support chat\n\n## Technical Stack\n- Framework: Next.js 14 (App Router) + React 18\n- Database: PostgreSQL with Prisma ORM\n- Auth: NextAuth.js v4 with Prisma adapter\n- Billing: Stripe Checkout (subscription mode) + Billing Portal\n- Background Jobs: BullMQ + Redis\n- Charts: Recharts for visualizations\n- UI: shadcn/ui components + Tailwind CSS 3\n- Icons: Lucide React\n- APIs: Plaid (banking), Stripe (payments)\n- Cloud Storage: AWS S3 for file uploads\n- PDF Generation: Abacus AI HTML2PDF API\n\n## Database Schema Summary\n\n### Core Models\n- **User**: Authentication, subscription tier, business info\n- **Transaction**: Income/expense records with Plaid sync\n- **Forecast**: 90-day cash flow predictions with advanced metrics (seasonality, trend, anomaly)\n- **LinkedAccount**: Plaid-connected bank accounts with sync cursors\n\n### Phase 3 Models\n- **ReportExport**: PDF/Excel report generation with S3 storage\n- **ApiKey**: Public API access with rate limiting\n- **Team**: Team workspaces with ownership\n- **TeamMember**: Team membership with roles\n- **TeamInvitation**: 7-day email invitations\n\n### Phase 4 Models\n- **AuditLog**: Tracks all data changes (create/update/delete/export)\n- **SharedDashboard**: Team dashboards with public share tokens\n\n### Phase 5 Models (Utilities)\n- Caching layer for dashboard, transactions, team data\n- Compliance exports for GDPR\n- Integration credential storage\n\n## Authentication & Authorization\n- Email/password auth with bcryptjs hashing\n- NextAuth.js sessions with JWT\n- Role-based access control: owner, manager, member, viewer\n- Permission checks for: read, create, update, delete, export, invite, view_audit_logs, manage_team\n- API key authentication for public data endpoints\n\n## File Structure\n```\n/home/ubuntu/cashflowiq/\n├── nextjs_space/\n│   ├── app/\n│   │   ├── api/\n│   │   │   ├── auth/ (NextAuth)\n│   │   │   ├──
